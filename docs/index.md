# 🧭 Bienvenue sur le Wiki **BESAFE**

Ce Wiki centralise toute la **documentation technique, les procédures et les schémas** du projet **BESAFE**.  
Il sert de **référence** pour l’équipe IT, l’exploitation, la formation et le suivi des livrables.

---

## 📚 Contenu principal

| Section | Description |
|----------|--------------|
| 🏗️ **Architecture** | Schémas physiques et logiques, VLAN, plan d’adressage, matrice de flux |
| ⚙️ **Services** | Nextcloud, GLPI, Centreon, Vaultwarden, Wazuh, etc. |
| 🖥️ **Systèmes & Réseaux** | ESXi, Stormshield, DNS/DHCP, PKI, sauvegardes |
| 🔒 **Sécurité** | Politiques, certificats, durcissement, supervision |
| 📘 **Runbooks & Procédures** | Installation, maintenance, dépannage, restauration |
| 🧾 **Projets & Changements** | RFC, suivi des tâches, PV de recette |
| 📎 **Annexes** | Modèles, gabarits, lexique, liens utiles |


## 🤝 Accès et collaboration

- 💡 **Objectif** : favoriser le partage de connaissances et la mise à jour continue de la documentation.  
- 🖊️ **Contributeurs** : toute personne impliquée dans le projet BESAFE peut ajouter ou corriger du contenu.  
- 🔍 **Transparence** : chaque modification est tracée et versionnée automatiquement.  
- 🧭 **Navigation** : utilisez la recherche ou le sommaire latéral pour retrouver facilement une page.
